# A² Compass — Complete Skill Prerequisite Tree

## How to Read This Document

Each skill has:
- **ID**: Unique code (domain letter + number) used in the database
- **Skill Name**: What students are learning
- **Grade Band**: Approximate grade equivalent (Pre-K through Grade 4+)
- **Prerequisites**: Which skills must be mastered (85%+) before this skill unlocks
- **Mastery Criteria**: What 85% looks like for this skill
- **Sample Activities**: Types of practice/assessment for this skill

**Prerequisite Rules:**
- Skills within a domain are generally sequential (A1 → A2 → A3)
- Some skills have **cross-domain prerequisites** (marked with ⚡)
- A skill marked "Ready to Learn" means all prerequisites are mastered
- "Entry Point" skills have no prerequisites — every student starts here

### Domain A: Print Concepts (6 levels)

These are the earliest literacy skills — understanding what a book IS and how print works.

| ID | Skill Name | Grade Band | Prerequisites | Mastery Criteria |
|----|-----------|------------|---------------|------------------|
| A1 | Book Orientation | Pre-K | *Entry Point* | Holds book right-side up, identifies front/back cover |
| A2 | Print Direction | Pre-K | A1 | Tracks left-to-right, top-to-bottom on a page |
| A3 | Word vs. Letter Distinction | Pre-K/K | A2 | Points to a "word" vs. a "letter" on a page |
| A4 | Word Boundaries (Spaces) | K | A3 | Identifies spaces between words; counts words in a short sentence |
| A5 | Sentence Concepts | K/1st | A4 | Identifies capital letters at start, period at end; understands a sentence is a complete thought |
| A6 | Text Features | 1st/2nd | A5 | Identifies title, author, page numbers, illustrations vs. text |

**Prerequisite Chain:** A1 → A2 → A3 → A4 → A5 → A6 (strictly linear)

**Sample Activities:**
- A1: "Tap the front of the book" (image-based)
- A2: "Drag your finger the way we read" (interactive arrow drawing)
- A3: "Circle the word / circle the letter" (tap-to-select)
- A4: "How many words in this sentence?" (counting tap)
- A5: "Find the capital letter" / "Find the period" (tap-to-select)
- A6: "Tap the title" / "Tap the author's name" (image regions)

### Domain B: Letter Recognition (6 levels)

Building from name awareness to full alphabet mastery.

| ID | Skill Name | Grade Band | Prerequisites | Mastery Criteria |
|----|-----------|------------|---------------|------------------|
| B1 | Letters in Own Name | Pre-K | *Entry Point* | Recognizes and names the letters in their first name |
| B2 | Uppercase Letter Recognition (Sequential) | Pre-K/K | B1 | Recites and identifies uppercase letters in alphabetical order |
| B2b | Uppercase Letter Recognition (Random Order) | Pre-K/K | B2 | Identifies all 26 uppercase letters by name **when shown in random order** — catches students who can sing the ABCs but can't name letters out of context |
| B3 | Lowercase Letter Recognition (Sequential) | K | B2b | Recites and identifies lowercase letters in alphabetical order |
| B3b | Lowercase Letter Recognition (Random Order) | K | B3 | Identifies all 26 lowercase letters by name **when shown in random order** |
| B4 | Upper/Lowercase Matching | K | B2b, B3b | Matches uppercase to corresponding lowercase (Aa, Bb, etc.) |
| B5 | Alphabetical Order | K/1st | B4 | Arranges letters in sequence; identifies what comes before/after |
| B6 | Letter Formation (Writing) | K/1st | B4 | Correctly forms both upper and lowercase letters (assessed by teacher observation, not auto-scored) |

**Prerequisite Chain:** B1 → B2 → B2b → B3 → B3b → B4 → B5 / B6 (B5 and B6 branch from B4)

**⚠️ Key Insight (Sandra's observation):** Many students can recite the alphabet song but cannot identify individual letters when shown randomly. B2→B2b and B3→B3b specifically catch this gap — a student who only knows the sequence will be flagged at the "Random Order" level.

**Sample Activities:**
- B1: "Find the letters in your name" (personalized letter cards)
- B2: "Sing and tap!" (letters appear in order, student names each one)
- B2b: "What letter is this?" (show uppercase **in random order**, tap name from 4 options)
- B3: "Name the small letters!" (lowercase in alphabetical sequence)
- B3b: "What letter is this?" (show lowercase **in random order**, tap name from 4 options)
- B4: "Match the big letter to the small letter" (drag-and-drop)
- B5: "What letter comes next? A, B, C, ___" (sequence completion)
- B6: Teacher-assessed checkpoint (tracing + free writing)

### Domain C: Phonological Awareness (11 levels)

The ability to hear and manipulate sounds in language — critical for reading.

| ID | Skill Name | Grade Band | Prerequisites | Mastery Criteria |
|----|-----------|------------|---------------|------------------|
| C1 | Environmental Sound Recognition | Pre-K | *Entry Point* | Identifies common sounds (doorbell, dog bark, phone ring) |
| C2 | Rhyme Recognition | Pre-K | C1 | Identifies whether two words rhyme (cat/hat = yes, cat/dog = no) |
| C3 | Rhyme Production | Pre-K/K | C2 | Produces a word that rhymes with a given word |
| C4 | Syllable Counting | Pre-K/K | C1 | Claps/counts syllables in spoken words (ba-na-na = 3) |
| C5 | Syllable Blending | K | C4 | Blends spoken syllables into a word ("pan" + "cake" = pancake) |
| C6 | Onset-Rime Recognition | K | C2, C4 | Separates initial sound from word ending (c-at, b-at) |
| C7 | Initial Sound Isolation | K | C6 | Identifies the first sound in a word ("What sound does 'fish' start with?") |
| C8 | Final Sound Isolation | K/1st | C7 | Identifies the last sound in a word |
| C9 | Phoneme Blending | K/1st | C7 | Blends individual sounds into a word (/c/ /a/ /t/ = cat) |
| C10 | Phoneme Segmenting | 1st | C9 | Breaks a word into individual sounds (cat = /c/ /a/ /t/) |
| C11 | Phoneme Manipulation | 1st/2nd | C10 | Substitutes, deletes, or adds sounds ("Say 'cat' without the /c/" = at) |

**Prerequisite Chain:**
```
C1 → C2 → C3
C1 → C4 → C5
C2 + C4 → C6 → C7 → C8
                C7 → C9 → C10 → C11
```

**Sample Activities:**
- C1: "What do you hear?" (play sound, tap matching picture)
- C2: "Do these rhyme?" (listen to 2 words, tap thumbs up/down)
- C3: "What rhymes with 'cat'?" (audio + picture options)
- C4: "Clap the parts!" (tap a drum for each syllable heard)
- C5: "Put it together!" (hear syllables, tap the whole word picture)
- C7: "What sound does this start with?" (audio word, tap letter sound)
- C9: "Sound it out!" (hear individual sounds, tap the blended word)
- C10: "Break it apart!" (hear word, tap each sound in order)
- C11: "Change the sound!" (hear "cat," change /c/ to /b/, what word?)

### Domain D: Phonics (11 levels)

Connecting letters to sounds — the bridge from awareness to reading.

| ID | Skill Name | Grade Band | Prerequisites | Mastery Criteria |
|----|-----------|------------|---------------|------------------|
| D1 | Letter-Sound Correspondence (common consonants) | K | B2b, C7 ⚡ | Produces the sound for common consonants (b, d, f, m, n, p, s, t) |
| D2 | Letter-Sound Correspondence (all consonants) | K | D1 | Produces sounds for all 21 consonants |
| D3 | Short Vowel Sounds | K | D1, B3b ⚡ | Identifies and produces short vowel sounds (a, e, i, o, u) |
| D4 | CVC Word Reading | K/1st | D2, D3, C9 ⚡ | Reads consonant-vowel-consonant words (cat, big, hot) |
| D5 | Consonant Digraphs | 1st | D4 | Reads ch, sh, th, wh, ck combinations |
| D6 | Consonant Blends | 1st | D4 | Reads bl, br, cl, cr, dr, fl, fr, gr, pl, pr, sl, sm, sn, sp, st, tr |
| D7 | Long Vowel Patterns (CVCe) | 1st | D4 | Reads silent-e words (cake, bike, rope, cute) |
| D8 | Vowel Teams | 1st/2nd | D7 | Reads ai, ay, ea, ee, ie, oa, oe, ue, oo patterns |
| D9 | R-Controlled Vowels | 2nd | D7 | Reads ar, er, ir, or, ur patterns |
| D10 | Diphthongs & Complex Vowels | 2nd/3rd | D8, D9 | Reads oi, oy, ou, ow, au, aw patterns |
| D11 | Multisyllabic Word Decoding | 3rd+ | D10 | Applies syllable division rules to decode longer words |

**Prerequisite Chain:**
```
B2b + C7 ⚡→ D1 → D2 ─┐
B3b ⚡→──────── D3 ─┘→ D4 → D5
               C9 ⚡→─┘     → D6
                         D4 → D7 → D8 → D10 → D11
                              D7 → D9 → D10
```

**Cross-Domain Prerequisites (⚡):**
- D1 requires B2 (must know letter NAMES before learning letter SOUNDS)
- D1 requires C7 (must isolate initial sounds before mapping them to letters)
- D3 requires B3 (must recognize lowercase letters for vowel identification)
- D4 requires C9 (must blend phonemes auditorily before blending in print)

**Sample Activities:**
- D1: "What sound does this letter make?" (show letter, tap sound from options)
- D4: "Read this word!" (show CVC word, tap matching picture)
- D5: "What word is this?" (sh-ip, ch-in — decode and match)
- D7: "Magic E!" (show word with/without e, identify which is which)
- D11: "Break and read!" (divide long word into syllables, then blend)

| E3 | First Grade Sight Words Set A (20 words) | K/1st | E2 | Reads: after, again, an, any, as, ask, by, could, every, fly, from, give, going, had, has, her, him, his, how, just |

| E5 | Second Grade Sight Words (23 words) | 1st/2nd | E4, D7 ⚡ | Reads: always, around, because, been, before, best, both, buy, call, cold, does, don't, fast, first, five, found, gave, goes, green, its, made, many, off |
| E6 | Third Grade+ Sight Words (25 words) | 2nd/3rd | E5, D8 ⚡ | Reads: about, better, bring, carry, clean, cut, done, draw, drink, eight, fall, far, full, got, grow, hold, hot, hurt, if, keep, kind, laugh, light, long, much |

- E6 requires D8 (vowel teams support more complex word recognition)

**Sample Activities:**

- "Find the word!" — hear a word, tap it among 4 word options
- "Sentence fill-in" — read sentence with blank, tap correct sight word
- Speed rounds for mastered sets (builds automaticity)

### Domain F: Vocabulary & Comprehension (7 levels)

From listening comprehension to critical reading — the ultimate literacy goal.

| ID | Skill Name | Grade Band | Prerequisites | Mastery Criteria |
|----|-----------|------------|---------------|------------------|
| F1 | Listening Comprehension (Basic) | Pre-K | *Entry Point* | Answers simple who/what questions about a read-aloud story |
| F2 | Listening Comprehension (Detailed) | Pre-K/K | F1 | Retells key events; answers where/when/why questions about stories |
| F3 | Vocabulary: Everyday Objects & Actions | K | F1 | Names and describes common objects, actions, and feelings (receptive + expressive) |
| F4 | Vocabulary: Categories & Relationships | K/1st | F3 | Sorts words into categories; identifies synonyms/antonyms; describes relationships |
| F5 | Reading Comprehension (Literal) | 1st | F2, D4 ⚡, E3 ⚡ | Reads a short passage and answers explicit who/what/where questions |
| F6 | Reading Comprehension (Inferential) | 2nd | F5 | Makes inferences; predicts outcomes; identifies main idea from a passage |
| F7 | Critical Comprehension | 3rd+ | F6, D11 ⚡ | Evaluates author's purpose; compares texts; distinguishes fact vs. opinion |

**Prerequisite Chain:**
```
F1 → F2 ────────────────────┐
F1 → F3 → F4                │
D4 + E3 ⚡→ F5 ←─── F2 ──────┘
                  F5 → F6 → F7
                        D11 ⚡→ F7
```

**Cross-Domain Prerequisites (⚡):**
- F5 requires D4 (must decode CVC words) AND E3 (must know first-grade sight words) — can't comprehend what you can't read
- F7 requires D11 (must decode multisyllabic words for grade 3+ texts)

**Sample Activities:**
- F1-F2: Listen to story (audio + illustrations), answer questions by tapping pictures
- F3: "Name that thing!" — show image, pick the word (or hear word, pick image)
- F4: "Which group does it belong to?" (drag word into category bucket)
- F5: Read short passage, answer comprehension questions (text-based for readers)
- F6: "What do you think will happen next?" / "What is this story mostly about?"
- F7: "Is this a fact or an opinion?" / "Why did the author write this?"

- G7: "Which has more?" (two groups, tap the bigger one)
- G8: "Count to 100!" (interactive number chart, fill in missing numbers)
- G10: "Who is third in line?" (picture of animals in a line, tap the 3rd one)
- G11: "Write the number!" (hear "seven," trace or write 7 on screen)
- G12: "Write what I say!" (hear "forty-three," write 43)

### Domain H: Number Operations (13 levels)

The journey from joining groups to division — the core of math fluency.

| ID | Skill Name | Grade Band | Prerequisites | Mastery Criteria |
|----|-----------|------------|---------------|------------------|
| H1 | Joining Groups (Pre-Addition) | Pre-K/K | G4 ⚡ | Combines two groups and counts the total (with objects) |
| H2 | Addition Within 5 | K | H1 | Solves addition problems where sum ≤ 5 |
| H3 | Separating Groups (Pre-Subtraction) | K | G4 ⚡, H1 | Removes objects from a group and counts what remains |
| H4 | Subtraction Within 5 | K | H3 | Solves subtraction problems within 5 |
| H5 | Addition Within 10 | K/1st | H2, G6 ⚡ | Solves addition problems where sum ≤ 10 |
| H6 | Subtraction Within 10 | 1st | H4, H5 | Solves subtraction problems within 10 |
| H7 | Addition Within 20 | 1st | H5, G9 ⚡ | Solves addition problems where sum ≤ 20 (with and without regrouping) |
| H8 | Subtraction Within 20 | 1st | H6, H7 | Solves subtraction problems within 20 |
| H9 | Place Value (Tens & Ones) | 1st | G8 ⚡, H7 | Understands that 34 = 3 tens + 4 ones |
| H10 | Addition/Subtraction Within 100 | 2nd | H8, H9 | Two-digit addition/subtraction with regrouping |
| H11 | Multiplication Concepts | 2nd/3rd | H10 | Understands multiplication as repeated addition; uses arrays |
| H12 | Multiplication Facts (0-10) | 3rd | H11 | Recalls multiplication facts through 10×10 |
| H13 | Division Concepts | 3rd | H11 | Understands division as equal sharing/grouping; relates to multiplication |
| H14 | Word Problems (Addition & Subtraction) | 1st/2nd | H8, F5 ⚡ | Solves real-world story problems requiring addition or subtraction within 20 — must **read** the problem and choose the operation |
| H15 | Word Problems (Multiplication & Division) | 3rd/4th | H12, H13, F6 ⚡ | Solves multi-step word problems involving multiplication and division |

**Prerequisite Chain:**
```
G4 ⚡→ H1 → H2 → H5 → H7 → H9 → H10 → H11 → H12 → H15
G4 ⚡→ H1 → H3 → H4 → H6 → H8 → H10     H11 → H13 → H15
              H5 → H6        H7 → H8 → H14
G6 ⚡→ H5    G9 ⚡→ H7   G8 ⚡→ H9    F5 ⚡→ H14   F6 ⚡→ H15
```

**⚠️ Key Insight:** Word problems are where math meets literacy — students must *read and comprehend* the problem before solving it. H14 requires both math fluency (H8) AND reading comprehension (F5). H15 requires inferential comprehension (F6) for multi-step problems. These cross-domain gates ensure students aren't failing at math when the real issue is reading.

**Cross-Domain Prerequisites (⚡):**
- H1 requires G4 (must match quantity to numeral before combining groups)
- H5 requires G6 (must recognize numerals 0-20)
- H7 requires G9 (must recognize numerals to 100 for sums above 10)
- H9 requires G8 (must count to 100 to understand place value)

**Sample Activities:**
- H1: "How many altogether?" (two baskets of fruit combine, count total)
- H2: "2 + 3 = ?" (visual objects + numeral equation, tap answer)
- H4: "5 - 2 = ?" (objects disappear, count remaining)
- H5-H8: Number line jumps, ten-frame activities, fact family houses
- H9: "Build the number!" (drag tens blocks and ones cubes to match number)
- H11: "3 groups of 4" (array visualization, count total)
- H13: "Share 12 cookies with 3 friends" (deal objects equally)

## 📊 CROSS-DOMAIN PREREQUISITE MAP (Summary)

These are the most important connections — they prevent students from attempting skills they're not ready for:

| Target Skill | Requires | Why |
|-------------|----------|-----|
| D1 (Letter-Sound) | B2b (Uppercase Random Order) + C7 (Initial Sounds) | Must recognize letters out of context AND hear sounds before connecting them |
| D3 (Short Vowels) | B3 (Lowercase Letters) | Vowels taught with lowercase letters |
| D4 (CVC Reading) | C9 (Phoneme Blending) | Must blend sounds orally before blending in print |
| E1 (Sight Words) | B2 (Uppercase Letters) | Must recognize letters before whole words |
| E2 (Primer Words) | D1 (Letter-Sound) | Beginning phonics supports word recognition |
| F5 (Reading Comp) | D4 (CVC Words) + E3 (1st Grade Sight Words) | Can't comprehend what you can't decode |
| F7 (Critical Comp) | D11 (Multisyllabic Decoding) | Grade 3+ texts require advanced decoding |
| H1 (Pre-Addition) | G4 (Quantity-Numeral Match) | Must connect quantities to numbers |
| H5 (Add Within 10) | G6 (Number Rec 0-20) | Must recognize the numbers involved |
| H9 (Place Value) | G8 (Count to 100) | Must understand the number sequence |
| J6 (Tell Time) | G6 (Number Rec 0-20) | Must read numbers on clock face |
| J10 (Count Coins) | H5 (Add Within 10) | Must add to total coin values |
| H14 (Word Problems: Add/Sub) | H8 (Sub Within 20) + F5 (Reading Comp) | Must read AND compute — catches students who fail math because they can't read the problem |
| H15 (Word Problems: Mult/Div) | H12 + H13 + F6 (Inferential Comp) | Multi-step problems require both math fluency and reading inference |

## 🎯 PLAYLIST ENGINE RULES

### Priority Order for Playlist Generation:
1. **Needs Practice** — Skills assessed between 50-84% (close to mastery, highest impact)
2. **Ready to Learn** — All prerequisites mastered but skill is "Not Started"
3. **Foundational Gaps** — Skills below 50% (may need prerequisite review first)

### Playlist Rules:
- **Default 5 active skills** on daily playlist (prevents overwhelm)
- **Teacher-adjustable cap per student**: Can set anywhere from 3-8 items depending on the student's stamina and focus level
- **Daily flexibility**: Teacher can flag "light day" (reduces to 3 items) or "power day" (up to 8 items) for any student on any day
- At least **1 skill per active domain** if student has gaps in multiple domains
- **Balance difficulty**: Mix 1-2 challenging items with 2-3 achievable items
- **Never show mastered skills** (85%+ → removed permanently from playlist)
- **Auto-promote**: When a skill is mastered, immediately check if new skills are unlocked (prerequisites met)
- **Spiral review**: Every 5th session, include one quick check on a recently mastered skill (not graded, just reinforcement)

### Daily Session Structure:
1. **Warm-Up** (2 min): Quick review of a recent mastery (confidence builder)
2. **Core Practice** (15-25 min): 3-4 playlist items, ~5-8 minutes each
3. **Challenge Round** (5 min): One "stretch" item from the next tier (optional, student-initiated)
4. **Celebration**: Summary of progress, stars earned, encouragement

### Teacher Override Capabilities:
- Add any skill to a student's playlist (even if prerequisites aren't met)
- Remove a skill from the playlist
- Reorder playlist priority
- Lock a skill as "focus area" (stays on playlist until teacher removes it)
- Mark a skill as "mastered" manually (bypassing assessment threshold)
- Set custom mastery threshold for individual students (e.g., 75% for a struggling student)
